# News Pulse — Event & Anomaly Analysis

## Skill Description
Analyze a specific news event, price movement, or fundamental anomaly affecting a holding, determine its root cause, and assess its impact on the investment thesis.

## Analytical Philosophy
Market reactions are often wrong in both direction and magnitude. The analyst's job is not to guess where the price will go next, but to correctly assess what the event means for business value.

## Analysis Framework

### Step 1 — Event Description
Be precise:
- What specifically happened? (price move %, earnings miss, management change, regulatory action, etc.)
- When did it occur?
- What was the market reaction (price + volume)?
- How did the broader market / sector perform over the same period?

### Step 2 — Root Cause Attribution
Rank causes by estimated contribution (% of total effect):

| Cause Category | Estimated Contribution | Evidence |
|---------------|----------------------|---------|
| Macro / market-wide (rates, sentiment, FX) | | |
| Sector-specific (regulatory, competition, cycle) | | |
| Company fundamentals (earnings, guidance, operations) | | |
| Market structure / liquidity / positioning | | |
| Pure sentiment / overreaction | | |

### Step 3 — Thesis Impact Assessment
For each component of the original investment thesis:
- Is this event **irrelevant** to the thesis?
- Does it **strengthen** the thesis (e.g., confirms moat)?
- Does it **weaken** the thesis (e.g., key assumption invalidated)?
- Does it **break** the thesis (e.g., structural damage to business)?

Assign: **No impact / Minor / Moderate / Major / Thesis-breaking**

### Step 4 — Forward-Looking Indicators
What data points should be monitored over the next 1–4 quarters to determine whether this event was a temporary blip or the start of a structural change?
- Metric 1: [what to watch] — threshold for concern
- Metric 2: [what to watch] — threshold for concern
- Metric 3: [what to watch] — threshold for concern

### Step 5 — Action Recommendation
- Does this event warrant a change in position size? (Add / Hold / Reduce / Exit)
- If adding: what price provides sufficient margin of safety given the new information?
- If reducing/exiting: what is the key reason the original risk/reward has deteriorated?
- What is the earliest date to revisit this assessment?

## Output Format
Complete all 5 steps. The action recommendation must be explicit and include a rationale tied to the thesis impact assessment, not just to price movement.
