# Quality Screen — Business Quality Assessment

## Skill Description
Assess the fundamental quality of a business using a structured scoring system, independent of current valuation. Quality first; price second.

## Quality Philosophy
Li Lu: "A high-quality business compounds value for you even if you overpay somewhat. A low-quality business destroys value no matter how cheaply you buy it."

## Scoring Framework (100 points total)

### Dimension 1 — Earnings Quality (20 points)
| Criterion | Max Points | Score | Evidence |
|-----------|-----------|-------|----------|
| FCF / Net Income > 85% | 8 | | |
| Stable or improving gross margin (5Y) | 6 | | |
| Low accruals ratio (< 5%) | 6 | | |

### Dimension 2 — Financial Strength (20 points)
| Criterion | Max Points | Score | Evidence |
|-----------|-----------|-------|----------|
| Net debt / EBITDA < 1.5× | 8 | | |
| Current ratio > 1.5× | 6 | | |
| Interest coverage > 5× | 6 | | |

### Dimension 3 — Growth Sustainability (20 points)
| Criterion | Max Points | Score | Evidence |
|-----------|-----------|-------|----------|
| Organic revenue CAGR > 8% (5Y) | 8 | | |
| Expanding addressable market | 6 | | |
| R&D / capex investment level supports future growth | 6 | | |

### Dimension 4 — Competitive Advantage (20 points)
| Criterion | Max Points | Score | Evidence |
|-----------|-----------|-------|----------|
| ROIC > 15% sustained (5Y) | 10 | | |
| Market share stable or growing | 6 | | |
| Pricing power evidence | 4 | | |

### Dimension 5 — Management Quality (10 points)
| Criterion | Max Points | Score | Evidence |
|-----------|-----------|-------|----------|
| Meaningful insider ownership (> 5%) | 4 | | |
| Consistent capital allocation discipline | 6 | | |

### Dimension 6 — Valuation Reasonableness (10 points)
| Criterion | Max Points | Score | Evidence |
|-----------|-----------|-------|----------|
| Not trading at > 2× historical premium | 5 | | |
| FCF yield > 3% (reasonable starting point) | 5 | | |

### Total Score Interpretation
| Score | Quality Tier | Interpretation |
|-------|-------------|----------------|
| 80–100 | **Exceptional** | Rare — commit meaningful research time |
| 65–79 | **High** | Watchlist candidate — monitor for entry price |
| 50–64 | **Average** | Only interesting if significantly discounted |
| < 50 | **Low** | Avoid unless extraordinary circumstance |

## Output Format
Complete the full scoring table with evidence for every criterion. Provide the total weighted score and quality tier. Include 3-sentence business quality verdict and one key risk that could lower the quality tier.
